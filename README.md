# ActiveResourceInspector

Allows you to inpsect your project's ActiveResource entities.

It prints ActiveResource entity's paths grouped by endpoint & auth type.

## Installation

```
gem 'active_resource_inspector', group: :development
```

## usage

```
rake resources:list
```
